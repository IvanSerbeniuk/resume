Eshop-Django

  • Live Demo
  • Source code

As a skilled Python developer, I have designed and implemented a sophisticated online shop using the Django framework.
The web app is hosted on a Digitalocean web-server running Ubuntu, Nginx, and Gunicorn, with the PostgreSQL database being used to store all relevant data.
The app is containerized using Docker and Docker-compose, ensuring easy portability and scalability.

To ensure a smooth and user-friendly experience, I utilized Bootstrap, HTML, CSS, and Jinja templates, along with Favicons.
To ensure user security, I integrated email verification for all users, ensuring a secure and streamlined user registration process.
Furthermore, the app integrates with Paypal for secure and convenient payments.
